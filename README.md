## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/lab-backend.git
```

Install all the required dependencies (for example)

```
$ cd classroom && npm i
```

Run all migrations

Then (at gateway)

```
$ npm start
```
